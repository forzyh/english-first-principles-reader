# 第20章　逻辑关系与连接

## 20.1 连接词为命题之间标边

句子之间可以存在原因、结果、转折、让步、条件、例证、递进、改述等关系。连接形式的功能，是让读者更快识别边的类型：

```text
A ──cause──> B
A ──contrast── B
A ──condition──> B
A ──evidence-for──> B
```

但连接词并不创造不存在的逻辑。therefore 只能标记说话者声称的推论，不能保证推论有效。

## 20.2 并列、从属与篇章连接副词

三类形式占据不同句法位置：

### 并列连词

> The data was incomplete, **but** we released the report.

but 连接两个相对对等的结构。

### 从属连词

> **Although** the data was incomplete, we released the report.

although 把一个命题标为背景让步条件，另一个成为主干。

### 连接副词

> The data was incomplete. **Nevertheless**, we released the report.

nevertheless 是第二句中的篇章副词，标记与前句的关系。它不能像 but 那样仅用逗号直接连接两个完整句：

> ✗ The data was incomplete, however we released the report.

可改为：

> The data was incomplete; however, we released the report.

或：

> The data was incomplete. However, we released the report.

## 20.3 原因关系有不同立场

> We stayed inside **because it was raining**.

说话者把下雨作为事件原因。

> **Since** you are already here, we can begin.

since 常把原因视为双方较容易接受的背景。

> He must be home, **because the lights are on**.

这里 because 不是现实事件因果，而是推理证据：灯亮支持“他在家”的结论。

读者需要判断连接的是世界中的事件，还是说话者的推理过程。

## 20.4 让步：预期被违反

> Although the task was difficult, she completed it on time.

让步关系隐含一个一般预期：任务困难通常会降低准时完成的可能；实际结果违反该预期。

```text
DIFFICULT(task)
expected → NOT COMPLETE-ON-TIME
actual   → COMPLETE-ON-TIME
```

but 也可表达反预期，但信息层级不同：

> The task was difficult, but she completed it on time.

两命题更对等；although 则把困难降为背景，让主句成为主要断言。

## 20.5 条件、必要条件与充分条件

> If the password is correct, the door opens.

逻辑上，password correct 被表达为足以触发开门的条件。但自然语言中的 if 常受常识限制，不一定声明严格双向等价。

> The door opens only if the password is correct.

only if 标记必要条件：开门要求密码正确。

> The door opens if and only if the password is correct.

双条件才明确表达必要且充分。日常交流通常不需要如此形式化，但在技术、法律和规范文本中，条件方向至关重要。

## 20.6 连接密度与隐性关系

初学写作常在每句前加 firstly、moreover、however、therefore。连接过密会让文本像显式标注的流程图，掩盖真正结构。若对象链、词汇衔接和段落组织已经清晰，逻辑关系可以部分隐含。

原则是：**在读者可能误判边类型时显式标记，在关系自然可恢复时允许结构本身工作。**

### 小练习

选择合适连接方式，并说明是事件因果、推理证据、让步还是条件：

- The roads were wet. It must have rained.
- The roads were wet. The race continued.
- You submit the form today. We can process it this week.
- The sample was small. The result should be interpreted cautiously.
