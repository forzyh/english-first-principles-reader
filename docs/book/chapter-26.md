# 第26章　可持续生长的学习路线

## 26.1 知识树比知识清单更容易生长

新知识若只是添加到清单，会与旧知识竞争记忆；若能挂到稳定父节点上，就会成为结构的一部分。例如学到 despite 时，不只是记“尽管”，而是挂到：

```text
篇章关系
└── 让步 / 反预期
    ├── although + clause
    ├── despite + noun phrase / -ing
    └── nevertheless + sentence-level link
```

这样，形式、补语类型和篇章功能同时进入网络。

## 26.2 三层学习单位

每个新表达至少记录三层：

### 形式层

```text
prevent + object + from + -ing
```

### 语义层

阻止某个参与者进入一个事件。

### 篇章层

常用于解释未发生结果或控制措施。

例句：

> The firewall prevented the malware from accessing the internal network.

只有三层齐全，知识才可迁移到新语境。

## 26.3 一周循环：输入—拆解—生成—反馈

### 第1—2天：高质量输入

选择略高于当前水平的真实材料，不追求覆盖大量生词。每次只精读一到两段，标出主干、引用链和事件元数据。

### 第3天：结构复述

不看原文，用语义图复述内容。比较自己选择的主语、冠词、时态和连接方式与原文有何差异。

### 第4—5天：受限生成

围绕同一内容写三版：口语版、说明版、正式版。限制自己必须使用本周目标构式。

### 第6天：错误归类

把反馈按第25章六层分类，避免只抄“正确答案”。记录错误触发条件与修复理由。

### 第7天：间隔回忆

不看笔记，重画本周新增的知识分支；用两个新例句验证是否真正掌握。

## 26.4 词汇应作为“可调用构式”学习

低效记录：

```text
insist = 坚持
```

可调用记录：

```text
insist on + NP/-ing
insist that + clause
meaning: maintain a demand/claim despite resistance
register: neutral to formal
example: She insisted on checking the figures herself.
contrast: insist that he leave / suggest that he leave
```

词汇量的真正价值，是能否在合适槽位和语域中调用。

## 26.5 建立个人错误数据库

每条记录包含：

| 字段 | 内容 |
| --- | --- |
| 原句 | 自己真实写过或说过的句子 |
| 错误层 | 主干、槽位、挂接、引用、元数据、篇章 |
| 触发原因 | 中文迁移、搭配未知、参照点跳转等 |
| 最小修复 | 尽量保留原意的版本 |
| 对比例句 | 同结构的正确新句 |
| 复测日期 | 间隔回忆计划 |

错误数据库比“错题本”更进一步：它寻找生成错误的机制，而不是收藏答案。

## 26.6 用生成能力衡量掌握

“看懂解释”不等于掌握。一个知识点通过以下四级测试才算稳定：

- **识别**：能在句子中发现；
- **解释**：能说明它为何这样编码；
- **对比**：能说明与邻近形式的差异；
- **生成**：能在新语境中自然使用，并根据焦点改写。

例如现在完成，不仅要认出 have + past participle，还要能根据参照点在 I lived、I have lived、I had lived 之间做有理由的选择。

## 26.7 最终目标：从规则调用转为结构直觉

第一性原理模型不是让你在说每句话前运行复杂算法。算法的作用像初学驾驶时的分步操作：通过大量有反馈的练习，层级、槽位和信息流逐渐自动化，最终形成语感。

真正可靠的语感不是无法解释的神秘直觉，而是大量结构模式压缩后的快速预测能力。你会自然感觉某个冠词不对、某个修饰语挂错、某个时态参照点跳了；需要时，又能退回本书模型说明原因。

## 26.8 全书闭环

```text
现实与意图
    ↓ 建模
对象 + 事件 + 关系
    ↓ 线性化
句法树 + 词汇接口
    ↓ 标注
时态 + 体 + 情态 + 语态
    ↓ 跨句更新
引用链 + 焦点 + 逻辑边
    ↓ 语境推断
实际交际意义
    ↓ 反馈
修正内部模型并再次生成
```

这不是一套读完即结束的教材，而是一棵可以不断挂接真实语言经验的树。任何新规则都应接受同一个问题：**它帮助说话者编码什么区别，又帮助听者恢复什么结构？**
