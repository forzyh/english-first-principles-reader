# 第22章　语境、歧义、礼貌与语域

## 22.1 语法只编码了意义的一部分

> Can you pass the salt?

字面结构询问能力，实际通常执行请求。听者之所以不会回答 “Yes, I can” 后毫无动作，是因为会结合餐桌场景、合作原则和社会惯例推断言外之意。

语用理解可以近似表示为：

```text
literal meaning
+ shared situation
+ assumed cooperative goal
+ social relationship
→ intended meaning
```

## 22.2 歧义不是系统失败，而是压缩结果

自然语言允许结构复用和省略，因此同一表面形式可能有多个解释。常见歧义层次包括：

- 词义歧义：bank 是银行还是河岸；
- 词类歧义：record 是名词还是动词；
- 挂接歧义：PP 或从句修饰谁；
- 作用域歧义：否定、量词、情态覆盖多大范围；
- 指代歧义：代词恢复哪个对象；
- 言语行为歧义：是在陈述、讽刺、请求还是拒绝。

理解者会同时利用概率、常识和上下文，而不是等到句末才启动推理。

## 22.3 量词作用域

> Every student read a book.

可能表示每个学生各读一本，不必相同；也可能在特定语境中指所有学生读同一本。形式逻辑可写为：

```text
∀ student ∃ book: READ(student, book)
∃ book ∀ student: READ(student, book)
```

自然语境通常偏向某一读法，但重要文本应通过改写消除风险：

> Every student read at least one book of their choice.

> All students read the same assigned book.

## 22.4 礼貌是对权利与负担的建模

直接命令：

> Send me the file today.

降低强制性：

> Please send me the file today.

> Could you send me the file today?

> Would it be possible to send me the file today?

形式越间接，通常给对方的选择空间越大，但也可能增加含糊。紧急、安全或职责明确的场景中，清晰直接反而更合适：

> Stop the machine immediately.

礼貌不是固定句型排行榜，而是根据权力、距离、负担、紧急度和文化预期调整语言距离。

## 22.5 语域决定允许多大压缩

口语对话依赖现场和共享上下文，因此可大量使用省略、代词和片段：

> Coming?

> Already did.

技术文档必须让未来、远程、不了解现场的读者也能恢复结构，因此倾向于显式对象、稳定术语和清楚条件：

> Restart the service only after the configuration file has been validated.

学术写作则需要区分事实、推断和限制：

> The results **suggest** that the intervention **may** reduce error rates, although the small sample limits generalization.

情态和让步结构在这里用于校准证据承诺。

## 22.6 正式不等于复杂

低质量“正式英语”常通过名词化、被动和长词堆叠制造距离：

> An evaluation of the implementation of the optimization was conducted.

更清楚的版本：

> The team evaluated how the optimization was implemented.

正式度应来自准确、可验证、边界清晰，而不是隐藏施事或增加不必要层级。

## 22.7 语法正确与交际恰当是两层判断

> I demand that you give me water.

语法可以正确，在餐厅中却通常不合适。

> Me and Sam went there.

在某些口语社群中自然，在正式标准写作中会改为 Sam and I went there。学习者需要同时掌握：

- 结构是否符合目标变体的语法；
- 表达是否适合当前语域和关系；
- 它在真实社群中可能携带什么身份信号。

### 第四部分检查点

你现在应能把跨句理解看成共享模型更新：

```text
创建对象 → 维持指代链 → 安排已知与新知
→ 标记命题关系 → 组织段落任务
→ 借助语境恢复言外之意
```

单句语法树到这里已经扩展为篇章语义图。
