# 总览：English First Principles 语法树

下图不是传统的“词类目录”，而是一张从意义出发的系统地图。根节点是“把现实编码成可恢复结构”；第一层分为句法树、事件元数据、篇章语义图和语境语用。此后的每一个知识点，都可以在这张树上找到位置。

![教材插图 3](/images/book/figure-03.png)


*图 0-2 English First Principles 总语法树。高清横版另附为独立海报。*

> **读图方法**：先沿着 A 分支建立句子骨架，再沿着 B 分支读取事件标签；跨句理解时进入 C 分支，最后由 D 分支利用语境完成消歧。不要从叶节点开始背诵。
