# 附录E　中英术语表

本表给出本书使用的工作定义。语言学传统之间可能采用不同术语或分析框架；判断时应以所解释的结构关系为准。

| English term | 中文工作译名 | 本书中的核心含义 |
| --- | --- | --- |
| adjacency | 线性邻接 | 两个形式在字符串中相邻，不保证直接组成单位 |
| agent | 施事 | 有意或主动发起事件的参与者 |
| agreement | 一致 | 主语与限定动词等形式之间的语法特征匹配 |
| ambiguity | 歧义 | 一个表面形式允许多个词义、结构、作用域或指代解释 |
| anaphora | 回指 | 用代词、定冠词等恢复前文对象或结构 |
| argument | 论元 | 由谓词意义选择、填入其角色槽位的成分 |
| aspect | 体 | 说话者把事件呈现为整体、过程、回看结果或持续过程的方式 |
| attachment | 挂接 | 一个修饰语或子树依赖于哪个父节点 |
| auxiliary | 助动词 | 承载时态、情态、完成、进行、被动、否定或疑问操作的功能词 |
| clause | 从句 / 小句 | 围绕一个述谓关系组织、可独立或嵌入的结构 |
| cleft | 分裂句 | 用 it is/was ... that/who... 显式标记对比焦点 |
| cohesion | 衔接 | 通过指代、连接、替代、省略和词汇关系连接文本表面 |
| coherence | 连贯 | 读者能否构造一致、有逻辑的篇章世界模型 |
| complement | 补语 | 由中心词选择、帮助完成其结构或意义的成分 |
| complementizer | 补语标记词 | that, whether, if 等提示嵌入从句边界和类型的形式 |
| constituent | 成分 | 在层级结构中作为一个单位运作的一组词 |
| control | 控制 | 高层成分决定非谓语结构隐含主语的关系 |
| counterfactual | 反事实 | 与已知现实相反或被说话者排除的可能世界 |
| deictic | 指示的 | 依赖说话时刻、地点或参与者坐标解释的表达 |
| determiner | 限定词 | 标记名词短语的识别、数量、指示或所属状态的形式 |
| discourse | 篇章 | 多个句子共同更新对象、事件与逻辑关系的层级 |
| dummy subject | 占位主语 | 填充句法主语位置但无具体指称的 it 或存在 there |
| end focus | 末端焦点 | 新信息或主要重音倾向于出现在结构末端 |
| end weight | 末端重量 | 较长、较复杂的成分倾向于后置 |
| event time (E) | 事件时间 | 事件实际或概念上发生的时间 |
| extraposition | 外置 | 用占位形式占据前部位置，把长从句移到句末 |
| finite | 限定的 | 能承载时态、主谓一致或独立从句操作层的动词/从句 |
| focus | 焦点 | 句子中被断言、对比或回答问题的核心新信息 |
| generic reference | 泛指 | 指向类别、一般成员或规律，而非某个具体实例 |
| given information | 已知信息 | 说话者假定听者当前可恢复或已激活的信息 |
| grammatical subject | 句法主语 | 占据从句主语位置、参与一致和倒装的成分 |
| head | 中心词 | 决定短语类别、内部选择和外部分布的核心节点 |
| information structure | 信息结构 | 已知、新知、主题、焦点如何映射到语序与构式 |
| lexical aspect | 词汇体 / 事件类型 | 谓词自身的时间形状，如状态、活动、完成过程、瞬时达成 |
| modifier | 修饰语 | 给对象、事件或命题增加可选属性或背景的成分 |
| modality | 情态 | 说话者对命题的可能性、必要性、许可、能力或意愿立场 |
| negation | 否定 | 对命题、事件或局部成分施加非真或排除操作 |
| new information | 新信息 | 当前句子要加入共享模型或设置为焦点的信息 |
| non-finite | 非限定的 | 不独立承载完整时态/一致、依赖高层结构解释的动词形式 |
| noun phrase (NP) | 名词短语 | 以名词或代词为中心、整体可占对象槽位的结构 |
| patient | 受事 / 受影响者 | 因事件发生改变、移动或受作用的参与者 |
| polarity | 极性 | 命题的肯定、否定及其相关环境 |
| predicate | 谓语 / 谓词 | 对主语建立状态、关系或事件的结构核心 |
| preposition phrase (PP) | 介词短语 | 以介词为中心、表达空间、路径或抽象关系的结构 |
| progressive disclosure | 渐进式披露 | 先建立可解析骨架，再在已有节点上追加较复杂信息的倾向 |
| pronoun | 代词 | 根据篇章可及性压缩并恢复对象引用的形式 |
| proposition | 命题 | 可被断言、否定、质疑或置于情态世界中的内容 |
| reference | 指称 / 引用 | 语言表达与篇章或现实对象之间的定位关系 |
| reference time (R) | 参照时间 | 说话者观察或组织事件的时间视点 |
| register | 语域 | 由场合、关系、媒介与社群规范决定的表达风格 |
| relative clause | 关系从句 / 定语从句 | 挂在名词节点上，并与从句内部角色建立绑定的子树 |
| remoteness | 距离性 | 过去形态编码的时间、现实或人际距离 |
| scope | 作用域 | 否定、情态、量词或焦点运算影响的结构范围 |
| semantic role | 语义角色 | 参与者在事件中的功能，如 agent、patient、recipient |
| speech time (S) | 说话时刻 | 当前话语发生的时间坐标 |
| subject complement | 主语补语 | 描述或识别主语状态、身份或结果的成分 |
| syntax | 句法 | 词与短语怎样组合成层级结构的系统 |
| tense | 时态 | 语法化地定位事件与说话/参照时点关系的形式 |
| theme | 主题物 / 话题 | 语义角色中被移动或传递的对象；篇章中也可指信息起点，需看语境 |
| transitivity | 及物性 | 谓语如何构形事件及其参与者、影响和对象 |
| valency | 动词价 / 配价 | 谓词开放多少个、何种语义与句法槽位 |
| voice | 语态 | 事件角色如何映射为主语、宾语及背景成分 |
| wh-movement | WH 移位 | 疑问成分与句内空位建立依赖并出现在从句前部的现象 |
