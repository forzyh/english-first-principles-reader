# 附录B　核心判断清单

## B.1 读一句话时

- 主句的限定谓语是什么？
- 主语中心词是什么？
- 动词当前词义开放哪些槽位？
- 哪些成分是必要补语，哪些是附加背景？
- 每个介词短语和从句挂在哪个节点？
- 时态的参照点在哪里？镜头在整体还是过程内部？
- 情态表达事实、推断、许可、义务还是礼貌？
- 否定与焦点的作用域有多大？
- 冠词和代词指向哪个篇章对象？
- 句子怎样接续前文已知信息？

## B.2 写一句话前

- 我想让读者首先注意哪个对象？
- 核心事件能否先写成最小主干？
- 选用的动词是否与补语模板匹配？
- 可数单数名词是否有明确限定？
- 长修饰是否应放在中心词之后？
- 时间状语是否与时态参照点一致？
- 需要直接断言，还是用情态降低承诺？
- 主动或被动哪个更符合话题连续？
- 代词是否可能有两个候选？
- 句末是否承载真正的新信息？

## B.3 改一句话时

先按以下顺序排查，避免一上来只换“高级词”：

```text
主干完整？
  ↓
动词槽位匹配？
  ↓
修饰挂接清楚？
  ↓
引用可恢复？
  ↓
时态/体/情态一致？
  ↓
篇章焦点与逻辑连贯？
  ↓
词汇与语域恰当？
```

## B.4 学一个新词时

记录：

```text
lemma / 基本形式
word class in this use / 当前词类
core frame / 核心语义图式
valency / 槽位
collocations / 搭配
register / 语域
contrast / 易混项
one personal example / 与自己有关的例句
```

## B.5 评估一个语法解释时

一个好解释应回答：

- 它编码了什么意义区别？
- 这个区别通过什么形式可观察？
- 它适用于哪些构式，边界在哪里？
- 是否把使用倾向误说成绝对规则？
- 是否能预测至少一个新例子？
- 是否能解释典型错误，而非只给正确答案？
